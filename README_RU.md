## Быстрый старт

Клонируйте репозиторий:

```bash
git clone https://github.com/danilfg/bank-test-platform.git
```

Перейдите в папку проекта:

```bash
cd bank-test-platform
```

Запустите через Docker:

```bash
docker compose up
```

## API

Пример базового API:
