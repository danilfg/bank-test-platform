## Быстрый старт

Клонируйте репозиторий:

```bash
git clone https://github.com/danilfg/bank
```

Перейдите в папку проекта:

```bash
cd bank
```

Запустите через Docker:

```bash
docker compose up
```

## API

Пример базового API:

```text
https://api.bank.easyitlab.tech
```

Пример запроса на логин:

```text
POST /auth/login
```
