## Quick Start

Clone repository:

```bash
git clone https://github.com/danilfg/bank-open-source.git
```

Enter project folder:

```bash
cd bank-open-source
```

Run with Docker:

```bash
docker compose up
```

## API

Example base API:

```text
https://api.bank.easyitlab.tech
```

Example login request:

