## Quick Start

Clone repository:

```bash
git clone https://github.com/danilfg/bank
```

Enter project folder:

```bash
cd bank
```

Run with Docker:

```bash
docker compose up
```

## API

Example base API:

```text
https://api.bank.easyitlab.tech
```

Example login request:

